# Booking Service
---
#### [Course work]

Booking Service is a service which helps you quicly book flight tickets

Used technologies:
 - .Net Core/C#
 - ASP.Net Core/Web API
 - Entity Framework Core
 - Telegram.Bot
 - PostgreSQL
 - Docker

Service consists of:
 - ASP.Net Core Web API
 - Client library
 - Telegram bot for clients

Available commands:
 - /help
---
### Development 
To start all services:
```
docker-compose up
```
